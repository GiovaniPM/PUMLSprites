GET /↩︎

Description: Home page (this help).
Method: GET
Request URL: /

POST /resize↩︎

Description: Resize image.
Method: POST
Request URL: /resize
Request Body:

{
  "image": "<base64_image>",
  "width": <new_width>,
  "height": <new_height>
}

Example:

curl --location '<server>:<port>/resize' \
--header 'Content-Type: application/json' \
--data '{
   "image": "<base64_image>",
   "width": <new_width>,
   "height": <new_height>
}'

POST /convert↩︎

Description: Convert image format.
Method: POST
Request URL: /convert
Request Body:

{
  "image": "<base64_image>",
  "format": "<new_format>"
}

[!TIP]
Note: <new_format> can be JPG, PNG, BMP, or GIF.

Example:

curl --location '<server>:<port>/convert' \
--header 'Content-Type: application/json' \
--data '{
   "image": "<base64_image>",
   "format": "<new_format>"
}'
